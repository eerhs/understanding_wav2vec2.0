# Understanding wav2vec 2.0: A Self-Supervised Approach to Speech Representation Learning

wav2vec 2.0 was developed by Facebook AI to leverage the vast amount of unlabeled speech data available in the world. Traditional automatic speech recognition (ASR) systems require thousands of hours of labeled (transcribed) audio to reach usable accuracy. Acquiring this data would be extremely expensive since such labeled data is scarce for most of the world's nearly 7,000 languages. wav2vec 2.0 addresses this by learning speech representations directly from raw audio without any labels, and using a small amount of labeled data for transcription.

## How to run:

1. Download Mini LibriSpeech datasets from https://www.openslr.org/31/

2. Copy dev-clean-2 and train-clean-5 folders to data/LibriSpeech

3. Run the python scripts and enjoy!!
